# 第二课：核心原理——不粘锅的学问

> 这是小白问得最多的问题：为什么肉一下锅就粘住了？

## 📺 视频 5：铁锅炒菜不粘的方法和思路（必看）

| 项目 | 内容 |
|------|------|
| 链接 | https://www.bilibili.com/video/BV1AU411f72o/ |
| UP主 | 阿喵才是up主本体 |
| 时长 | 25:45 |
| 播放量 | 7.6万 |

📝 **笔记要点：**
**不粘锅核心法则：热锅凉油**

原理：
1. 锅烧热 → 铁表面温度升高 → 油倒入后瞬间形成润滑层
2. 食材下锅时遇到高温 → 表面迅速凝固焦化（梅纳反应）→ 自然脱离锅底
3. 凉锅下食材 = 食材还没凝固就已经粘上锅底了

**正确的操作流程：**
```
Step 1: 大火空烧锅 → 到冒青烟（约200°C）
Step 2: 倒油 → 晃匀 → 让油布满锅壁
Step 3: 油热了（微微冒烟）→ 下食材
Step 4: 食材下锅后不要马上翻 → 等10-15秒让它表面凝固
Step 5: 轻轻晃锅 → 食材能滑动 → 说明没粘 → 开始翻炒
```

**如果已经粘了怎么办？**
- 小火 → 加点水 → 盖上盖子焖一下 → 粘住的部分自己脱落
- 或者大火快炒时粘了 → 别用力铲 → 加点油 → 等一会再翻

✍️ **一句话总结**：锅要够热，油要够滑，肉要等一面煎定了再翻。

怎么判断锅够热了？
- 方法1：手放在锅上方10cm处，感到**灼热感**（不要摸！）
- 方法2：滴一滴水进锅里，水珠像荷叶上的水珠**滚来滚去**（莱顿弗罗斯特效应）→ 温度到了
- 方法3：锅开始**微微冒青烟** → 温度到了（约200°C）

**不同食材的最佳入锅温度：**
| 食材 | 锅温 | 判断方法 |
|------|------|---------|
| 肉类（猪肉/鸡肉） | 高温（冒烟） | 下锅有"滋啦"声 |
| 鸡蛋 | 中温（不冒烟） | 油微动，蛋液入锅不粘 |
| 蔬菜 | 中高温 | 下锅有短暂水汽声 |
| 鱼 | 高温 | 下锅前油要烧到冒烟 |

🔧 **动手任务**：用滴水法练习判断锅温，直到能准确分出"不够热"、"刚好"、"太热"三个档。

| 项目 | 内容 |

| UP主 | 曹哥做饭可以 |

📝 **笔记要点：**
炒肉不粘的三步法：
1. **腌制**：肉切好→加一勺生抽+半勺淀粉+一勺油→抓匀（这叫"上浆"）
2. **热锅热油**：锅冒烟→倒油→油热→倒出热油→加凉油（这叫"热锅凉油"）
3. **快速滑炒**：肉下锅→快速划散→变色就捞出

为什么腌制有用？
- 淀粉在肉表面形成保护膜 → 肉汁不流失 → 更嫩
- 油裹住肉 → 阻隔肉和锅的直接接触 → 不粘

✍️ **记住**：但凡炒肉丝肉片，先腌制10分钟，再热锅凉油炒，基本不会翻车。

📝 **笔记要点：**
**新手最容易犯的4个粘锅错误：**
1. ❌ **锅没烧够热就下油** → 锅不够热，油膜形成不了
2. ❌ **食材太湿就下锅** → 水遇到油会爆，食材容易粘
3. ❌ **刚下锅就急着翻** → 食材表面还没凝固，一翻就破
4. ❌ **火太小** → 温度不够，食材出水→变成"煮"不是"炒"

**正确做法对照：**
1. 锅烧到冒烟再倒油
2. 食材沥干水分或用厨房纸吸干
3. 下锅后等15-20秒再翻
4. 炒肉用大火，炒菜用中大火

# 第三课：核心技能——火候怎么把握

> 小白最头疼的问题：什么时候用大火？什么时候用小火？

## 📺 视频 9：铁锅炒菜技巧——怎么炒出锅气和火

| 项目 | 内容 |
|------|------|
| 链接 | https://www.bilibili.com/video/BV1Et4y1G7fw/ |
| UP主 | 段小段美食教程 |
| 时长 | 1:54 |
| 播放量 | 13.5万 |

📝 **笔记要点：**
**火候的三大类型及用法：**

**大火（猛火）：**
- 适用：爆炒蔬菜、炒肉丝肉片、需要"锅气"的菜
- 特征：食材入锅"滋啦"声大、水分迅速蒸发
- 关键：动作要快，火大容易糊

**中火：**
- 适用：煎鱼、煎豆腐、炒蛋、炖菜收汁
- 特征：油微微滚动，食材慢慢上色
- 关键：耐心，不要急着翻

**小火（微火）：**
- 适用：炖肉、焖煮、炸东西、煎厚肉
- 特征：油面平静，偶尔冒泡
- 关键：时间长，急不得

**新手速记表：**
```
🍃 绿叶菜 → 大火快炒  （2-3分钟）
🥩 肉丝肉片 → 大火滑炒（2-3分钟）
🐟 煎鱼 → 中小火慢煎（每面4-5分钟）
🥚 炒蛋 → 中火     （1-2分钟）
🍲 炖肉 → 大火烧开转小火慢炖（40-60分钟）
🥔 煎土豆 → 小火慢煎  （每面5-7分钟）
```

🔧 **动手任务**：连续一周，每次炒菜时刻意关注自己用的火候大小，看菜谱上的火候建议自己做对比。

## 📺 视频 10：菜熟没熟，怎么判断？（综合笔记）

这个没有专门的B站长视频，但综合各UP主的经验，整理如下：

📝 **判断食物是否熟了的通用方法：**

**看颜色：**
- 肉：生肉红色/粉色 → 熟了白色/褐色（猪肉）、白色（鸡肉）
- 绿叶菜：生硬挺→熟了变软、颜色变深
- 鸡蛋：蛋液透明→凝固变白/金黄

**看状态：**
- 肉：用筷子戳最厚的地方 → 没有血水渗出 = 熟了
- 蔬菜：用锅铲切一下 → 能轻松切断 = 熟了
- 鱼：筷子能轻松插入鱼身最厚处 = 熟了

**靠计时（新手最靠谱的方法）：**
```
蔬菜类：
  叶菜（青菜/菠菜）：大火约1-2分钟
  根茎类（土豆/胡萝卜）：中火约3-5分钟（建议先焯水）
  
肉类：
  肉丝肉片：大火约2-3分钟
  鸡胸肉整块：中火每面5-7分钟
  鱼：中火每面4-5分钟（视厚度调整）
  
蛋类：
  炒蛋：中火约1-2分钟
  煎蛋：中小火约2-3分钟
```

✍️ **新手建议**：
1. 不确定的时候 **用筷子夹一块切开看**，最直接
2. 宁可多炒30秒，也别吃生的（但绿菜炒太久就黄了，要注意）
3. 肉类不确定 → 用温度计插最厚处（75°C以上安全）

# 第四课：健康烹饪——低油少盐的技巧

## 📺 视频 11：清炒系列——少油做菜的方法

结合多位UP主（老东北美食、阿蔡美食雕刻等）的内容整理：

📝 **低油炒菜四法则：**

**法则一：选对锅**
- 铁锅用久了油膜形成后，比不粘锅用油还少
- 锅够滑，油可以用得更少

**法则二：水炒代替油炒**
- 对于某些蔬菜：热锅 → 加少量水 → 下菜 → 盖上焖30秒 → 开盖大火收干
- 这样只用一勺油，甚至不用油

**法则三：利用食材自身油脂**
- 炒五花肉、鸡腿肉 → 先干煸出自身的油 → 再炒菜
- 这样菜的油来自肉，不用额外加

**法则四：调料减盐不减味**
- 用生抽代替盐 → 更鲜、更咸，用量更少
- 用葱姜蒜、花椒、干辣椒提味 → 减少盐的依赖
- 用醋、柠檬汁代替部分盐 → 酸味能突出咸味

**低盐天然调料清单：**
```
基础：生抽、老抽、醋、料酒
提鲜：葱、姜、蒜、洋葱、香菇
增香：花椒、干辣椒、八角、桂皮
清香：香菜、芹菜、葱花、芝麻
```

# 第六课：实践篇——经典家常菜谱（低油少盐版）

> 以下菜谱按小白友好度排序

## 菜谱 1：番茄炒蛋（最基础，必会）

| 配料 | 用量 |
|------|------|
| 鸡蛋 | 2-3个 |
| 番茄 | 2个（成熟多汁） |
| 油 | 1.5勺 |
| 盐 | 半茶匙 |
| 葱花 | 少许 |

**做法：**
1. 番茄切块，鸡蛋打散加少许盐
2. 铁锅烧热 → 倒油（1勺）→ 油热下蛋液
3. 蛋液凝固就划散 → 盛出备用
4. 锅里补半勺油 → 下番茄 → 中火炒出汁（用铲子压一压）
5. 番茄软了 → 倒回鸡蛋 → 加盐 → 翻炒均匀
6. 撒葱花 → 出锅

**小白要注意的：**
- 鸡蛋不要炒太久，凝固了就出锅 → 不然就老了
- 番茄要炒出"红油"才好吃 → 可以加一丁点糖提鲜
- 想少油：蛋液里加一勺水，炒出来更嫩，油可以减半

## 菜谱 2：蒜蓉炒青菜（学火候）

| 配料 | 用量 |
|------|------|
| 青菜（油麦菜/生菜/菠菜） | 300g |
| 蒜 | 3-4瓣（切碎） |
| 油 | 1.5勺 |
| 盐 | 半茶匙 |

**做法：**
1. 青菜洗净沥干（水分越少越好）
2. 锅大火烧到冒烟 → 倒油
3. 下蒜蓉 → 爆香（约10秒）
4. 下青菜 → 大火快炒
5. 菜变软 → 加盐 → 翻匀 → 关火出锅

**小白要注意的：**
- 菜必须**沥干水分**！水多=煮菜=难吃
- 全程大火，动作要快，从下锅到出锅不超过3分钟
- 菜量多的话分两次炒，不要一次塞太多
- 吃之前尝尝咸淡，不够再加，不要一次加太多盐

## 菜谱 3：青椒肉丝（学腌肉和滑炒）

| 配料 | 用量 |
|------|------|
| 瘦肉（猪里脊） | 150g |
| 青椒 | 2个 |
| 生抽 | 1勺 |
| 淀粉 | 半勺 |
| 油 | 2勺 |

**做法：**
1. 肉切丝 → 加生抽+淀粉+半勺油 → 抓匀腌制10分钟
2. 青椒切丝
3. 锅大火烧到冒烟 → 倒1勺油 → 滑入肉丝 → 快速划散 → 变色捞出
4. 不用洗锅 → 加半勺油 → 下青椒丝 → 大火炒1分钟
5. 倒回肉丝 → 加半勺生抽 → 翻匀 → 出锅

**小白要注意的：**
- 肉丝要切均匀，不能一块大一块小
- 腌制是关键，不要省这一步
- 炒肉动作要快，肉变色就捞出来
- 铁锅炒肉丝不粘的秘诀：**锅够热、油够多、肉别急着翻**

## 菜谱 4：清蒸鱼（最少油盐）

| 配料 | 用量 |
|------|------|
| 鱼（鲈鱼/鲳鱼/多宝鱼） | 1条 |
| 姜 | 几片 |
| 葱 | 几段 |
| 蒸鱼豉油 | 1.5勺 |
| 油 | 1勺 |

**做法：**
1. 鱼洗净，两面划几刀 → 塞姜片
2. 蒸锅水烧开 → 放鱼 → 大火蒸8-10分钟
3. 蒸好倒掉盘里的水（去腥）
4. 撒上葱丝 → 淋蒸鱼豉油
5. 另起锅烧1勺油到冒烟 → 浇在葱上（滋啦一声）
6. 开吃

**小白要注意的：**
- 蒸鱼不腥的关键：**倒掉蒸出来的水**
- 蒸的时间：鱼眼鼓出、筷子能插入鱼身最厚处 = 熟了
- 没有蒸鱼豉油 → 用生抽+一点点糖代替
- 油只用在最后泼油那一步，实际摄入很少

## 菜谱 5：可乐鸡翅（新手友好炖菜）

| 配料 | 用量 |
|------|------|
| 鸡翅中 | 8-10个 |
| 可乐 | 1罐（330ml） |
| 姜 | 几片 |
| 生抽 | 1勺 |
| 油 | 半勺 |

**做法：**
1. 鸡翅两面划刀 → 冷水下锅煮开 → 捞出洗净
2. 铁锅烧热 → 半勺油 → 下鸡翅中火煎到两面金黄
3. 倒可乐 → 加姜片+生抽 → 大火烧开
4. 转中小火炖15分钟
5. 转大火收汁（汤汁浓稠裹住鸡翅即可）

**小白要注意的：**
- 先焯水去血水，不然腥
- 煎鸡翅时不要急着翻，一面煎定了再翻
- 收汁时看着锅，容易糊！
- 可乐本身有糖，不需要额外加糖

# 常见问题速查表

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| 炒肉粘锅 | 锅不够热 / 肉太湿 / 油太少 | 热锅凉油、肉擦干、多放一点油 |
| 菜炒出来水汪汪 | 火太小 / 菜没沥干 | 用大火、菜沥干再下锅 |
| 菜炒黄了 | 炒太久 / 火太大 | 绿叶菜控制在3分钟内 |
| 肉炒老了 | 火太大太久 | 大火快炒，变色就出锅 |
| 盐放多了 | 手抖 | 加一点点糖或醋中和，或者加点水稀释 |
| 铁锅生锈 | 用完没烘干 / 没抹油 | 每次洗完烧干抹油 |
| 锅冒烟很大 | 油温过高 | 关小点火，或者先离火冷却一下 |
| 葱姜蒜糊了 | 油温太高下配料 | 油微热时下葱姜蒜，不要等冒烟 |
| 菜不入味 | 盐放晚了 | 炒菜过程中加盐，别等快出锅才放 |
| 鱼皮粘锅破了 | 锅不够热 / 鱼太湿 | 鱼擦干、锅烧到冒烟再下油、一面煎定了再翻 |
