# 蒜苔炒肉
> 收录日期：2026-05-27

中式家常基本功菜。肉丝嫩滑 + 蒜苔脆嫩，一软一脆层次分明。蒜苔自带清甜蒜香，是整道菜的灵魂。

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 猪瘦肉（里脊/梅花肉） | 200g，切丝 |
| 蒜苔 | 250g，去头尾切4cm段 |
| 姜 | 3片，切丝 |
| 蒜 | 2瓣，切片（可选） |

**腌肉料：** 生抽1汤匙、料酒1茶匙、淀粉1茶匙、食用油半茶匙

**调味料：** 生抽1汤匙、老抽半茶匙、蚝油半汤匙、白糖一小撮、盐适量

### 🔪 准备步骤

1. **腌肉** — 肉丝加生抽、料酒、淀粉抓匀，最后淋油封住，腌制10分钟。💡 *顺纹切片逆纹切丝，上浆锁水是肉嫩的关键*
2. **处理蒜苔** — 去头尾切段，掐一下根部：掐得动留，掐不动扔

### 🥘 烹饪步骤

1. **焯蒜苔** — 水开加盐和几滴油，蒜苔入锅30-40秒变翠绿，捞出过凉水沥干
2. **滑肉丝** — 锅烧热多倒油，油温五六成下肉丝，筷子快速划散，变色（约30秒）盛出，留底油
3. **爆香炒合** — 姜丝（+蒜片）小火爆香 → 转中大火下蒜苔翻炒30秒 → 下肉丝 + 生抽老抽蚝油白糖 → 大火翻炒30秒融合
4. **出锅** — 尝味，通常酱油蚝油咸度已够，不够补盐

### 💡 关键要点

- **肉丝嫩滑三件套**：上浆 + 滑油 + 不过度烹饪
- **蒜苔不要焯太久**：30-40秒去生味保脆绿，过凉水锁色
- **这道菜的主角是蒜苔不是肉**：蒜苔要脆要绿有存在感
- **变体**：猪肉换鸡胸/牛肉同样上浆处理；加几朵木耳增加脆感层次；加干辣椒爆香变微辣版更下饭

### 📺 参考

- 家常快手菜，基础扎实无需参考视频

# 香葱煎蛋
> 收录日期：2026-05-27

升级版葱煎蛋，从"能吃"到"好吃"的完整拆解。外层微焦酥香，内里嫩滑蓬松，三个鸡蛋一把葱搞定。

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 鸡蛋 | 3个（室温，不要冰的） |
| 小葱/香葱 | 3-4根，切细葱花 |

**调料：** 盐1/4茶匙、白胡椒粉一撮、食用油2汤匙

**可选加料：** 虾皮1茶匙 / 火腿丁2汤匙 / 萝卜干碎1汤匙

### 🔪 准备步骤

1. **打蛋液** — 鸡蛋 + 盐 + 白胡椒，筷子快速搅打到表面起细密泡沫（打入空气 = 蓬松）
2. **加葱** — 葱花加入蛋液轻轻搅匀。💡 *一半打入蛋液，一半留到最后撒表面，能吃到两种葱香*

### 🥘 烹饪步骤

**方案一：整张蛋饼（练翻锅）**

1. **热锅热油** — 中大火烧锅到明显热气 → 倒油晃匀 → 油微微冒烟下蛋液（"哗啦"一声是好信号）
2. **煎制** — 不要动！等底部定型（约1分钟），边缘翘起、晃锅蛋饼能滑动 → 翻面
3. **翻面** — 铲子辅助 + 颠锅，或先滑入盘中再倒扣回锅，另一面30秒-1分钟至金黄出锅

**方案二：炒蛋块（零失败）**

1. 蛋液倒入，等10秒底部凝固 → 铲子从外向内推，让未凝固蛋液流到锅底
2. 重复推进几次，基本凝固后翻面炒散成大块 → 微焦出锅

### 💡 关键要点

- **蓬松秘诀**：蛋液打到起泡 + 油要够热冒烟 + 别着急翻
- **粘锅克星**：铁锅用"热锅凉油法"——烧热→倒油晃锅→倒掉→重新倒凉油
- **油温不够蛋会吸油**：看到油微微冒烟再下蛋液
- **中餐馆招厨师第一道就考这道**：鸡蛋葱盐油，没地方藏拙
- **灵魂吃法**：煎蛋铺热饭上淋一圈生抽

# 返沙番薯
> 收录日期：2026-05-23

**来源：** [超详细返沙番薯教程](https://www.bilibili.com/video/BV1bd2JYFE7m) — UP主：@潮菜叶飞一日三餐
**参考：** [年夜菜必备：反沙番薯](https://www.bilibili.com/video/BV1s64y1f7PJ) — UP主：@胖江生活美食
**技法参考：** [大师的菜·反沙芋头](https://www.bilibili.com/video/BV1XJ411m7iU) — UP主：@大师的菜（非遗传承人）

> 反沙是潮汕传统甜品技法，核心是把糖浆炒到结晶返沙，裹在炸熟的番薯表面，外酥里糯，糖霜不腻。

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 红薯（番薯） | 1-2 个（约 400-500g） |
| 白砂糖 | 100-120g |
| 水 | 50-60ml |
| 食用油 | 适量（炸用） |
| 葱花（可选） | 少许 |

### 🔪 准备步骤

1. **处理番薯** — 去皮，切成滚刀块或手指粗的条（大小均匀，方便熟成一致）
2. **清洗去淀粉** — 切好后用清水冲洗两遍，沥干水分，用厨房纸吸干表面（防溅油）

### 🥘 烹饪步骤

1. **炸番薯** — 锅中放宽油，油温五成热（约 150°C，筷子放进去冒细泡），下番薯块中小火慢炸 5-6 分钟至熟透、表面微黄，捞出控油。💡 *不要大火，否则外焦里生*
2. **复炸（可选）** — 油温升至七成热，下番薯复炸 30 秒至表面酥脆，捞出控油
3. **熬糖浆** — 锅中倒掉油，洗净锅。放入糖和水，开中火搅拌至糖完全融化
4. **炒糖关键** — 继续熬煮糖浆，不要搅拌（会返砂），等糖浆从大泡变密集小泡、呈浓稠状时立即关火。💡 *判断标准：糖浆用铲子提起后滴落呈线状，最后几滴有拉丝感*
5. **返沙** — 关火后静置约 30 秒让糖浆稍微降温，倒入炸好的番薯，迅速**不停翻炒**。随着温度下降，糖浆会从透明变白，逐渐结晶返沙，均匀裹在番薯表面
6. **出锅** — 撒少许葱花增香（正宗潮汕做法），翻匀即可出锅

### 💡 关键要点

- **番薯要炸透但别炸过头** — 用筷子能轻松扎透就是熟了。复炸一下更酥脆
- **糖水比例约 2:1** — 糖 100g + 水 50ml 起步，新手按这个比例不容易翻车
- **熬糖别搅！** — 糖融化后不要搅动糖浆，否则提前结晶，返沙效果差
- **关火后降温再下番薯** — 刚关火的糖浆温度太高，下番薯会延缓结晶；等 30 秒左右再倒番薯翻炒，效果最好
- **全程不停翻** — 返沙过程就是翻炒让糖浆降温均匀结晶，别停手
- **同理可得** — 这个技法同样适用于反沙芋头、反沙山药、反沙腰果
- **翻车补救** — 如果糖不返沙一直黏糊，说明熬得不够（水分太多），回火再熬一下；如果糖浆变硬结成块，说明熬过头了，下次缩短熬糖时间

# 苦瓜炒肉
> 收录日期：2026-05-23

**来源：** [苦瓜炒肉怎么做才好吃？大厨教你技巧](https://www.bilibili.com/video/BV1b64y1t7Jy) — UP主：@超子美食
**参考：** [教你苦瓜炒肉家常做法](https://www.bilibili.com/video/BV1bLZAYkEcQ) — UP主：@天天相见厨房阿鹏

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 苦瓜 | 1根 |
| 猪瘦肉（里脊或前腿） | 150-200g |
| 大蒜 | 3-4瓣，切片 |
| 生姜 | 2-3片 |
| 小葱 | 1根，切段 |

**调料：** 酱油、蚝油、盐、鸡精、食用油

💡 *适配说明：原视频用料酒腌肉、白胡椒粉和白糖调味，已省略；淀粉勾芡用少量蚝油替代。*

### 🔪 准备步骤

1. **处理苦瓜** — 对半切开，用勺子挖去瓤和白膜（白膜最苦，刮干净），切片约 3mm 厚
2. **去苦涩** — 烧一锅水，水开后放少许盐和油，下苦瓜焯水 30秒，捞出过凉水沥干
3. **腌肉** — 瘦肉切薄片（逆纹切更嫩），加少许酱油、蚝油抓匀，最后封一点食用油锁水，腌制 10 分钟
4. **调汁** — 小碗中混合酱油、蚝油、盐、鸡精、2勺水，搅匀备用

### 🥘 烹饪步骤（铁制炒锅）

1. **滑肉** — 铁锅烧热至冒烟，倒油晃匀润锅，倒出热油再加冷油（热锅冷油防粘）。油温四五成热下肉片，快速划散炒至变色，盛出备用
2. **爆香** — 锅中留底油，下姜片、蒜片爆香
3. **合炒** — 下焯好的苦瓜，大火翻炒约 30秒
4. **调味** — 倒入调好的料汁，翻炒均匀
5. **回肉** — 倒入滑好的肉片，大火快速翻炒 15-20秒
6. **出锅** — 撒葱段，翻炒两下即出锅

### 💡 关键要点

- **苦瓜去苦三连招：** 刮净白膜 → 焯水（加油盐）→ 过凉水
- **铁锅防粘：** 一定要热锅冷油，锅烧到冒烟再下油润锅
- **全程大火快炒：** 所有食材都预处理过，最后合炒不超过一分钟
- **控油：** 滑肉油量不用多，肉片变色就盛出，避免老油

# 青椒肉丝
> 收录日期：2026-05-23

**来源：** [我做的"青椒肉丝"很好吃！步骤清晰详细](https://www.bilibili.com/video/BV1xS4y167Qe) — UP主：@美食强

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 青椒 | 2-3个 |
| 猪瘦肉 | 200g |
| 大蒜 | 3瓣，切末 |
| 生姜 | 少许，切丝 |

**调料：** 酱油、蚝油、盐、鸡精、食用油

💡 *适配说明：原视频用料酒、淀粉腌肉，已省略；用蚝油替代勾芡效果。*

### 🔪 准备步骤

1. **切肉** — 瘦肉切丝（顺纹切不易断），加少许酱油、蚝油抓匀腌制 10分钟
2. **切青椒** — 青椒去籽切丝
3. **调汁** — 碗中混合酱油、蚝油、盐、鸡精、少许水搅匀

### 🥘 烹饪步骤（铁制炒锅）

1. **滑肉丝** — 热锅冷油，下肉丝快速划散炒至变色，盛出备用
2. **炒青椒** — 锅中留底油，下姜蒜爆香，倒入青椒丝大火翻炒至断生（约 1分钟）
3. **合炒** — 倒回肉丝，淋入调好的料汁，大火翻炒均匀约 20秒即可出锅

### 💡 关键要点

- 肉丝不要炒太久，变色就盛出，否则会老
- 青椒不要炒太软，保持脆感
- 控油：青椒不太吸油，炒菜油量比平时少一半

# 菜椒炒肉
> 收录日期：2026-05-23

**来源：** [青椒炒肉做法参考](https://www.bilibili.com/video/BV1395jzHED6) — UP主：@刘小欠教做菜7

> 菜椒（甜椒/灯笼椒）口感清甜不辣，和肉片搭配非常下饭。

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 菜椒（甜椒） | 2个 |
| 猪瘦肉 | 200g |
| 大蒜 | 3瓣，切片 |
| 生姜 | 2片 |

**调料：** 酱油、蚝油、盐、鸡精、食用油

### 🔪 准备步骤

1. **切肉** — 瘦肉切薄片，加酱油、蚝油抓匀腌制 10分钟
2. **处理菜椒** — 去籽去白筋，切块或切丝
3. **调汁** — 酱油、蚝油、盐、鸡精、少许水搅匀

### 🥘 烹饪步骤（铁制炒锅）

1. **滑肉** — 热锅冷油，下肉片炒至变色盛出
2. **炒菜椒** — 锅中留底油，下姜蒜爆香，倒入菜椒大火翻炒至微软（约 1分钟）
3. **合炒** — 倒回肉片，淋料汁，大火翻炒均匀出锅

### 💡 关键要点

- 菜椒比青椒水分多、口感脆甜，不需要炒太久
- 肉片腌制时加点油锁水，口感更嫩

# 炒生菜（蚝油生菜）
> 收录日期：2026-05-23

**来源：** [厨师长教你："蚝油生菜"的家常做法](https://www.bilibili.com/video/BV1bS4y1r7jR) — UP主：@美食作家王刚R

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 生菜 | 2棵 |
| 大蒜 | 4瓣，切末 |

**调料：** 蚝油、酱油、盐、食用油

### 🔪 准备步骤

1. **洗生菜** — 掰开叶片，清水洗净沥干
2. **调汁** — 蚝油 + 少许酱油 + 水搅匀备用

### 🥘 烹饪步骤（铁制炒锅）

1. **焯水（推荐）** — 烧一锅水，水开后放少许油盐，下生菜烫 15秒立刻捞出装盘。💡 *这样比直接炒更脆嫩、更控油*
2. **或直接炒** — 热锅少油，下蒜末爆香，放生菜大火快炒 30秒，淋蚝油汁翻匀出锅

### 💡 关键要点

- 生菜出水多，大火快炒控好时间
- 焯水法油量更少，适合低油饮食

# 炒菜心
> 收录日期：2026-05-23

**来源：** [炒了几十年的菜心，有可能是错的？](https://www.bilibili.com/video/BV1fu4m1u7o3) — UP主：@厨师徐嘉乐

### 📋 配料表

| 食材 | 用量 |
|------|------|

3. **合炒** — 锅中放蒜片爆香，倒回土豆和肉片，加酱油、盐、鸡精大火翻炒均匀
4. **出锅** — 撒葱段，翻匀即出锅

### 💡 关键要点

- 土豆泡水去淀粉后煎出来更脆
- 土豆一定要煎熟（用筷子能扎透），不熟会有生味
- 也可以用微波炉先把土豆叮 3分钟代替煎，更控油

# 蒜苔炒干鱿鱼
> 收录日期：2026-05-24

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 蒜苔（蒜薹） | 250g |
| 干鱿鱼 | 1-2 只（约 80-100g） |
| 姜 | 3-4 片，切丝 |
| 大蒜 | 3 瓣，切片 |
| 青红椒（可选配色） | 各半根，切丝 |

**调料：** 酱油、蚝油、盐

💡 *适配说明：原菜谱可能使用料酒去腥，已省略；干鱿鱼充分浸泡换水即可去腥。*

### 🔪 准备步骤

1. **泡发干鱿鱼（关键）** — 干鱿鱼提前用冷水浸泡 **4-6 小时**（或隔夜），中途换 2-3 次水。泡至回软后，撕去表面薄膜和中间的透明骨片
2. **切花刀** — 鱿鱼内侧（有纹路面）斜刀 45° 切十字花刀，再切 3-4cm 的块。切花刀是为了遇热卷曲、更入味
3. **处理蒜苔** — 蒜苔洗净，切去老根和顶部花苞，切 4-5cm 段
4. **焯水鱿鱼** — 烧一锅水，水开后下鱿鱼块，焯 15-20 秒，鱿鱼卷曲立即捞出沥干（焯久了会老）

### 🥘 烹饪步骤（铁制炒锅）

1. **爆香** — 热锅少油，下姜丝、蒜片爆香
2. **炒鱿鱼** — 下焯好的鱿鱼卷，大火快炒 30 秒，沿锅边淋少许酱油炒出香味，盛出备用
3. **炒蒜苔** — 补一点点油，下蒜苔大火翻炒约 1-2 分钟至颜色深绿、表面微皱
4. **合炒** — 鱿鱼倒回锅中，加蚝油、少许盐，快速翻炒均匀（约 30 秒）
5. **出锅** — 撒入青红椒丝翻两下，关火出锅

### 💡 关键要点

- **干鱿鱼泡发是成败关键** — 泡够时间，中途换水。赶时间可用温水泡（缩短到 2-3 小时），但口感稍差
- **鱿鱼不能久炒** — 焯水和炒的总时间控制在 1 分钟内，否则变橡皮
- **蒜苔先炒后合** — 鱿鱼和蒜苔熟成时间不同，分开炒再合炒才能各自到位
- 这道菜**味道鲜香**，干鱿鱼本身的咸鲜味和蒜苔的清甜很搭
- 💡 *补充提示：干鱿鱼泡发后若觉得腥，焯水时水里加几片姜*

# 清炒秋葵
> 收录日期：2026-05-24

### 📋 配料表

| 食材 | 用量 |
|------|------|
| 秋葵 | 250-300g |
| 大蒜 | 3-4 瓣，切片 |
| 干辣椒（可选） | 1-2 个，切段 |

**调料：** 盐、蚝油

### 🔪 准备步骤

1. **处理秋葵** — 秋葵洗净，用盐搓洗表面绒毛，冲洗干净
2. **去蒂** — 切去秋葵顶端硬蒂，注意不要切太深露出内部籽
3. **焯水（关键去涩）** — 烧一锅水，加少许盐和几滴油。水开后下秋葵，焯 **1-2 分钟**，捞出过凉水，沥干
4. **切段** — 斜刀切 3-4cm 段（或整根保留，看个人喜好）

### 🥘 烹饪步骤（铁制炒锅）

1. **爆香** — 热锅少油，下蒜片、干辣椒段爆香
2. **快炒** — 下秋葵，大火快速翻炒约 1 分钟
3. **调味** — 加少许盐、蚝油翻炒均匀，立即出锅

### 💡 关键要点

- **秋葵一定要焯水** — 去除涩味和部分黏液，口感更清爽。不焯水直接炒会很黏且带涩
- **焯水时水里加盐和油** — 能保持秋葵翠绿色泽
- **全程大火快炒** — 秋葵已经焯熟，炒太久会软塌出水
- 秋葵本身味道清淡，蒜香和蚝油的咸鲜是主要味道来源
- 秋葵内部黏液其实是**可溶性膳食纤维**，对肠胃好，不用怕

- 不需要参考视频，这道菜非常简单
