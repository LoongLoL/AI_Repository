## 🗺️ 学习路线建议

### 🟢 阶段一：语言基础（1-2个月）

1. **环境搭建**：安装 Visual Studio 2022（Community 版免费），创建第一个控制台项目
2. **基本语法**：变量、数据类型、运算符、流程控制（if/else、switch、for、while）
3. **方法**：参数、返回值、可选参数、ref/out 参数
4. **面向对象**：类与对象、封装、继承、多态、抽象类与接口
5. **常用类型**：数组、List、Dictionary、String、StringBuilder、DateTime

> 📌 **推荐教程入门：** C#语言入门详解 或 2025版C#零基础超详细教程
> 📌 **配套练习：** 每学完一章，至少独立编写3-5个小程序（如计算器、猜数字游戏、学生信息管理系统）

---

### 🟡 阶段二：进阶核心（1-2个月）

1. **委托与事件**：Action、Func、自定义 Delegate、事件驱动编程
2. **Lambda 表达式**：理解箭头函数语法，熟练使用 LINQ
3. **LINQ**：集合查询、Where、Select、GroupBy、Join、聚合函数
4. **泛型**：泛型类、泛型方法、泛型约束（where T : class）
5. **异常处理**：try/catch/finally、自定义异常
6. **文件IO**：StreamReader/StreamWriter、序列化（JSON/XML）
7. **异步编程**：async/await、Task、Task.Parallel、CancellationToken

> 📌 **推荐教程进阶：** 新版C#从入门到精通全套视频教程（180全集）
> 📌 **配套练习：** 实现一个异步文件下载器、LINQ 数据查询工具

---

### 🟠 阶段三：Web 开发（2-3个月）

1. **ASP.NET Core 基础**：项目结构、中间件管道、依赖注入、配置系统
2. **Web API / Minimal API**：RESTful 设计、路由、模型验证、Swagger
3. **Entity Framework Core**：Code First、迁移、CRUD、关系映射、性能优化
4. **身份认证**：JWT、ASP.NET Core Identity、角色权限管理
5. **前端集成**：Vue3/React + .NET Core 前后端分离开发
6. **部署**：Docker 容器化、Nginx 反向代理、Linux 部署

> 📌 **推荐教程 Web：** ASP.NET Core 9.0 从入门到实战 + 杨中科 .NET Core 教程
> 📌 **配套项目：** 电商后台管理系统 / 博客系统 / 在线考试系统

---

### 🔴 阶段四：高级与架构（持续学习）

1. **设计模式**：单例、工厂、策略、观察者、装饰器、仓储模式
2. **微服务架构**：gRPC、RabbitMQ 消息队列、Ocelot 网关、Consul 服务发现
3. **性能优化**：Span\<T\>、Memory\<T\>、对象池、BenchmarkDotNet 基准测试
4. **AOT 编译**：理解 Native AOT 原理，发布自包含可执行文件
5. **Blazor**：用 C# 替代 JavaScript 开发交互式 Web UI
6. **AI 集成**：Semantic Kernel、Microsoft Agent Framework 多智能体开发

> 📌 **推荐教程高级：** .NET 8 + Vue3 电商项目实战 + .NET 9 AOT 新特性
> 📌 **配套项目：** 微服务电商系统 / AI Agent 应用 / Blazor 全栈项目
