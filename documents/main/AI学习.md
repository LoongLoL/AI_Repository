**核心概念记录：**
1. ReAct = Reason（推理）+ Act（行动）
2. 具体的工作循环是什么？
   - 接收到问题 → 思考需要什么信息 → 采取行动（查资料/执行代码）→ 观察结果 → 再思考 → 循环直到完成
3. 这个过程和人类解决问题的方式有什么相似之处？

**画一张图：**
```
用户提问
  ↓
AI 思考：我需要做什么？
  ↓
AI 行动：调用工具 / 查资料 / 执行代码
  ↓
观察结果
  ↓
是否完成任务？
  ├─ 是 → 回复用户
  └─ 否 → 继续思考 → 继续行动
```

# 💡 学习建议

## 每天怎么学

```
每天 1-2 小时，分三步：
1. 看视频（40分钟）—— 边看边做笔记
2. 动手练（30分钟）—— 照着视频做一遍
3. 整理笔记（15分钟）—— 用自己的话总结
```

## 遇到不懂的怎么办

1. **先自己搜** — B 站、Google、GPT 问一圈
2. **再看一遍视频** — 很多时候第一次没看懂，第二次就懂了
3. **动手试错** — 很多概念做了就懂了
4. **还不行就问** — 但先做前三步

## 最重要的几条原则

1. **不要只看不练** — 看过10个视频不如亲手做1个项目
2. **先抄后创** — 跟着视频做一遍，再做自己的
3. **不懂也要试** — 代码跑不起来也是一种学习
4. **保持节奏** — 每天1小时比周末10小时更有效
